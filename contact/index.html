Security reports

Use the same address for security issues: bloomlateral@gmail.com.

For a suspected data exposure, describe the affected page and steps to reproduce without sending personal task content.

Data requests

Task data stays in your browser by default. Export it from the File menu, or remove it through your browser’s site-data controls.

If you enabled optional Google Drive sync, use the in-app account controls to disconnect Google, revoke access everywhere, or delete Taskliner’s encrypted Drive files. Those actions are separate from deleting the local document. Disconnecting Discord completion sharing is also available in the Discord settings dialog.

See the Data & Sync guide and Privacy Policy for step-by-step details.

For a deletion or privacy request by email, write to bloomlateral@gmail.com. Include whether the request is about browser data, Google account connection, Drive sync files, or Discord sharing. Do not send task titles or notes unless they are necessary.
