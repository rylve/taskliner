ログアウトしてもこの端末のタスクは残ります。同期データ削除はすべての端末に影響します。

復旧キーを使う
  1. 同期済み端末でTasklinerを開く
  2. 「アカウント」→「別の端末を同期する」を開く
  3. QRをこの端末のカメラで読むか、招待コードを下に入力する

既存端末と同じ確認コードなら、この端末で同期を開始します。

Branch list

Tap a row to open the branch inside it. Tap the pencil icon to edit it in place with the keyboard.

Use ☷ to switch between branch and full-outline views

Use ↕ Reorder and drag the handle. Change parents from ⋯ → Move

Add and details

Add items repeatedly to the current branch with Add item

Open due date and notes from ⋯ → Details

Close or Back returns to the same branch

Browser Back closes the open surface before leaving the branch

Tap ⋯ or long-press a row to get Details, Move, Complete, and Delete.

Welcome to Taskliner

On phones, you manage tasks by walking branches.

  • Use Add item to write in the current branch
  • Tap a row to open that branch
  • Use ⋯ for details, move, complete, and delete

File

Import and export tasks as JSON files.

Categories

When enabled, top-level items are treated as categories.

Categories themselves cannot be completed and show no progress gauge.

In due-date order, category rows stay as headings while their tasks sort by due date.

Opening a task under a category shows the category name above the detail title.

Top-level items archived while categories were off still work as usual.

データについて

Tasklinerは登録不要で使えます。

タスクはあなたの端末内のサイトデータとして保存されます。

Googleアカウントを接続すると、端末間でタスクを同期できます。ブラウザのデータが消えても、同じGoogleアカウントを接続すれば復元できます。
