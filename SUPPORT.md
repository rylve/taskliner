# Support

## Official hosted app

For ordinary bugs and concrete product feedback about [taskliner.app](https://taskliner.app), open a GitHub Issue using the Bug Report template when possible.

## Security

Security issues must go through the private channels in [SECURITY.md](./SECURITY.md), not public Issues.

## Self-hosting

Self-hosted deployments are supported on a best-effort documentation basis only.

There is **no guarantee** of individualized setup help, debugging, or operational support for self-hosted instances.

## Forks and derived services

Forks and third-party deployments are **not** covered by official support.

## Limits

- Reply timelines are not guaranteed
- Individual data recovery is not guaranteed
- Losing every device, passkey access, and recovery file for an encrypted workspace means the maintainer cannot restore your tasks
